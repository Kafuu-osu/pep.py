## 需要
Python 3.8

## 初始化子模块
```
git submodule init
git submodule update
```

## 安装依赖
```
pip38 install -r requirements.txt
```


## 编译
```
python38 setup.py build_ext --inplace
```


## 运行
```
python38 pep.py 
```

## 修改配置文件
```
vim config.ini
```
